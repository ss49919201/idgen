## idgen

`idgen` is ID generator.

Generate UUID or ULID with a easy command.

### Install idgen

```
$ go install github.com/s-beats/idgen@latest
```

### Usage

#### Generate UUID

```sh
$ idgen uuid
UUID: 86aa212d-3333-4faf-848f-08cdb56326da
```

#### Generate ULID

```sh
$ idgen ulid
ULID: 01GFDQJEDE7YQXKCRMQQ9NFJPD
```
